# Plan&Eat

## Project overview

Plan&Eat is a family meal planning service for building a 7-day weekly menu, managing a shared dish library, and generating a shopping list from selected dishes.

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase

## Local setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a local environment file based on `.env.example` and fill in your local values.
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000).

## Environment variables

Real environment values are stored locally and must not be committed to the repository. Use `.env.example` as the reference for required variables and keep actual secrets in your local `.env.local`.
